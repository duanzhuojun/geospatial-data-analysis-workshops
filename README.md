# Geospatial Data Analysis Workshops

This repository contains materials for hands-on workshops on thunderstorm studies, geospatial analysis, and data mining techniques.

## Website

Visit the workshop website: [https://duanzhuojun.github.io/geospatial-data-analysis-workshops](https://duanzhuojun.github.io/geospatial-data-analysis-workshops)

## Local Development

To run the site locally:

1. Install Jekyll:
   ```bash
   gem install bundler jekyll
   ```

2. Clone the repository:
   ```bash
   git clone https://github.com/duanzhuojun/geospatial-data-analysis-workshops.git
   cd geospatial-data-analysis-workshops
   ```

3. Install dependencies:
   ```bash
   bundle install
   ```

4. Run the local server:
   ```bash
   bundle exec jekyll serve
   ```

5. Open your browser to `http://localhost:4000/geospatial-data-analysis-workshops`
